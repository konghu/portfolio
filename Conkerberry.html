Conkerberry

D3 + Gulp + UIKit

This is an open source project I did with two other classmates, under the supervision of a program manager from Microsoft Azure. I am the main developer. This tool is intended to solve the user's pain point when a large amount of nodes is needed. In order to create Microsoft Azure Service Cluster, users had to manually construct JSON files. This process is very error-prone and time-consuming.

We believed a browser-based JSON generator would properly address this issue. We come up with an interactive canvas that visualizes relations among nodes and facilitates users to construct node relations with just drag and drop.
